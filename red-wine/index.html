Red Wine Quality Prediction

Predicting wine quality using machine learning and physicochemical features.

Overview

A predictive analytics project focused on identifying key chemical properties influencing red wine quality. Multiple models including Logistic Regression, Random Forest, Support Vector Machines, and Neural Networks were developed and compared, emphasizing feature selection, tuning, and performance evaluation.
