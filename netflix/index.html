Data storytelling using Netflix datasets and audience-driven visual analysis.

Overview

This project analyzed multiple Netflix datasets to uncover trends in global content popularity and regional viewing behavior. Deliverables included a Jupyter Notebook, PowerPoint presentation, and narrative designed for a non-technical audience, emphasizing storytelling clarity and ethical data use.
