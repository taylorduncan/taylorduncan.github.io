Gold Price Forecasting

Forecasting gold market behavior using macroeconomic indicators and machine learning.

Overview

This project applies machine learning techniques to forecast gold market behavior using macroeconomic and sentiment-related indicators. Models including Linear Regression, Random Forest, Support Vector Regression, and Neural Networks were evaluated using RMSE, MAE, and R². The work includes EDA, feature importance, ethical assessment, and implementation planning.

Methods
