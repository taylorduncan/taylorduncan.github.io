Big Data Pipeline (NiFi • HDFS • Hive • Spark • HBase)

End-to-end ingestion and processing pipeline across the Hadoop ecosystem.

Overview

A full big-data ingestion and processing pipeline built using Apache NiFi for data ingestion, HDFS for distributed storage, Hive for querying structured tables, Spark for transformation and computation, and HBase for NoSQL storage. The project included architecture documentation, screenshots, and validation of distributed workflows.

NiFiHDFSHiveSparkHBaseDocker

Pipeline Flow

  1. Ingest data with NiFi processors and manage flow configuration
  2. Write raw/processed data to HDFS and verify file placement
  3. Create Hive managed/external tables for structured querying
  4. Transform and analyze data with Spark jobs
  5. Validate outputs and document system behavior and results
